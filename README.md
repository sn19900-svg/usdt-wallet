# محفظة USDT - TRC-20

تطبيق Android لإدارة USDT على شبكة Tron (TRC-20)

## الميزات
- ✅ إنشاء محفظة جديدة بـ 12 كلمة (BIP39)
- ✅ استيراد محفظة موجودة
- ✅ عرض الرصيد (USDT + TRX)
- ✅ إرسال USDT لأي عنوان TRC-20
- ✅ استقبال عبر QR Code
- ✅ سجل المعاملات
- ✅ تشفير AES-256 للمفتاح الخاص

## البناء
```bash
git clone https://github.com/sn19900-svg/usdt-wallet
cd usdt-wallet
chmod +x gradlew
./gradlew assembleDebug
```

أو ادفع لـ GitHub وسيبني تلقائياً عبر Actions.

## ملاحظة
- الشبكة: Tron TRC-20 فقط
- لا ترسل ETH أو BNB أو شبكات أخرى لهذه المحفظة
